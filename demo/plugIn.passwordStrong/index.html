二：简单规则

  1. 长度6-20
  2. 只允许输入数字+字母，不允许出现中文或者符号
  3. LV0 ： 所有不符合条件的
  4. LV1 ： 纯字母、纯数字
  5. LV2 ： 字母+数字
具体计分规则见内网wss

三：插件参数介绍

弱密码集合:
weak : ["asdfgh","qwerty","zxcvbn","Password","Passwd","Woaini","Iloveyou","Woaiwojia","521521","5201314","7758521","1314520","1314521","520520","201314","211314","7758258","147258369","159357","123456","1234567","12345678","123456789","654321","123123","123321","123abc"]

提示语:(对象排列:{"0":"友情提示","1":"长度错误时提示语","2":"密码为弱密码时提示语","success":"密码可用时候的提示语"})： tips : {"1":"长度不对","2":"弱密码，请修改","success":"可用"}

判断是否可用
根据密码输入dom里的"data-type"判断，0为不可用，1为可用 


使用方法：
$('#m_passwordStrong1').passwordStrong({
	"inpId" : null,		//输入框的id，可不填
	"type" : true,		//类型，是否出现 点击可出现密码，默认true，不要则填false
	"eyesClass" : null,	//眼睛class 点击查看密码
	"color" :{"tips":["#ccc","red"]},	//颜色{"提示文字" : ['默认颜色','强度颜色']},PS:后期可继续扩展控制其他颜色
	"weak" : []			//弱密码
	tips : {"0":"友情提示","1":"长度错误时提示语","2":"密码为弱密码时提示语","success":"密码可用时候的提示语"}//提示语 
});
